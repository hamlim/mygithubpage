CSS Bugs 1: Inline-Blocks and that darn space

How to remove that extra space between inline-block elements

: October 18th, 2016

If you are an active web developer and have ever worked with CSS before, you know that it can be a really pesky langauge with a lot of gotchas and quirks. One of these known quirks is the spacing rendered between inline-block elements. This is an odd bug that happens because of the whitespace between the blocks. This spacing can be seen in the pen embedded below:

See the Pen Inline Block Element spacing by Matt Hamlin (@matthamlin) on CodePen.

In the above example because there is a new line and some tabs between each inline-block element, the browser renders space between each element. Sometimes this is a really finicky bug where you might spend hours looking for where that padding or margin was added to that one element.
