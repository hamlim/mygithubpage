Speaking of only loiading the code you need there was a concept that was actually quite interesting that was introduced by one of the web developers on the messenger.com, he was talking about the normal autoprefixer solution that most web devs use. However messenger takes the next step, they only ship css that can be used on the users browser. So instead of shipping 5+ lines of autoprefixed css they only ship the one line that is needed for your browser. This seems like quite a simple idea but it sounds more like a really exciting idea. Imaging being able to deliver one fifth the css that you ship when building websites.
